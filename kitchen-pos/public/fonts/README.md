# Font Files

Place your Programme font files here:

- `Programme-Regular.woff2` (weight: 400)

If you have the fonts in other formats (TTF, OTF), you can convert them to WOFF2 using tools like:
- https://cloudconvert.com/ttf-to-woff2
- https://transfonter.org/

Alternatively, if you only have one weight, update `/app/layout.tsx` to reference just that file.
